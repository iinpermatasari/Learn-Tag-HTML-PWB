# Tag-Tag dalam HTML
Belajar beberapa tag dasar dalam HTML

## Contoh dan Fungsi setiap tag
```html
1. tag <html> berfungsi sebagai tag pembuka dalam membuat dokumen HTML.
2. tag <head> memuat informasi meta tentang dokumen, seperti judul.
3. tag <title> berfungsi untuk menetapkan judul dari halaman web.
4. tag <body> mendefinisikan body atau isi dari halaman web dan berisikan seluruh konten yang ditampilkan halaman web.
5. tag <h1> sampai <h6> merupakan jenis tag heading yang berfungsi untuk membuat heading pada halaman.
6. tag <hr> berfungsi untuk membuat garis horizontal pada halaman, biasanya untuk memisahkan konten.
7. tag <p> berfungsi untuk mendefinisikan paragraf.
8. tag <b> berfungsi untuk membuat teks bold (tebal).
9. tag <i> fungsinya untuk membuat teks italic (miring).
10. tag <u> berfungsi untuk menggaris bawahi teks.
11. tag <a> berfungsi untuk mendefinisikan hyperlink untuk menghubungksn ke halaman web lain. Tag HTML hpyperlink tampil sebagai bagian yang bisa di-klik oleh pengunjung.
12. tag <img> berfungsi untuk menyisipkan gambar pada halaman.
13. tag <!–…–> merupakan HTML comment tag yang berfungsi untuk menuliskan komentar pada dokumen HTML.
14. tag <ol> berfungsi untuk membuat daftar yang berurutan.
15. tag <ul> berfungsi untuk membuat daftar yang tidak berurutan.
16. tag <li> mendefinisikan suatu item di dalam daftar.
17. tag <form> berfungsi untuk membuat form pada suatu halaman web.
18. tag <label> mendefinisikan label untuk elemen tag input.
19. tag <input> mendefinisikan berbagai jenis input.
20. tag <button> tag HTML button berfungsi untuk membuat tombol.
21 tag <footer> mendefinisikan bagian footer halaman web.
```
## Sedikit Contoh dan Fungsi Atribut dalam HTML

```html
1. type = Menentukan jenis dari sebuah elemen (biasanya pada <input>). Ini memberitahu browser apakah kotak tersebut untuk teks, password, email, atau tombol.
2. id = Memberikan nama unik pada satu elemen tertentu.
3. name = Memberikan identitas data saat formulir dikirim ke server (backend).
4. for = Digunakan pada elemen <label> untuk menghubungkannya dengan elemen input (berdasarkan id). Jadi, kalau kamu klik teks labelnya, kursor otomatis masuk ke kotak inputnya.
5. placeholder = Memberikan teks petunjuk atau "bayangan" di dalam kotak input sebelum pengguna mengetikkan sesuatu. Teks ini akan hilang otomatis saat mulai diketik.
6. &copy; = Digunakan untuk memunculkan simbol Copyright (©) di halaman web. Karena simbol tersebut tidak ada langsung di keyboard standar, kita menggunakan kode tersebut agar browser menampilkannya dengan benar.
```
